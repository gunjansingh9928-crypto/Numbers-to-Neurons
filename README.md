# Numbers to Neurons | Stamatics Summer Project

An in-depth exploration of the advanced mathematical foundations of Artificial Intelligence, focusing on the first-principles implementation of deep learning frameworks. This project bridges the gap between theoretical mathematics (linear algebra, calculus, and optimization) and the programmatic execution of neural architectures.

---

## Project Overview

* **Timeline:** May 2026 – Present  
* **Organization:** Stamatics, IIT Kanpur  
* **Objective:** To demystify "black-box" machine learning by building, training, and debugging neural pipelines from scratch using only fundamental mathematical libraries.

---

## Key Features & Technical Execution

### 1. First-Principles Implementations
* **Custom Computational Graphs:** Built modular forward and backward propagation pipelines from scratch using **Python** and **NumPy**.
* **Mathematical Optimization:** Implemented gradient-based learning algorithms, tracking weight updates through exact partial derivatives and chain-rule calculus.
* **Tensor Engineering:** Programmed low-level tensor preprocessing, matrix manipulation, and shape alignment without relying on high-level deep learning frameworks (like PyTorch or TensorFlow).

### 2. Architectural Deep Dives
* **Convolutional Frameworks:** Mapped multi-dimensional calculus to the structural execution of **Convolutional Neural Networks (CNNs)**, including spatial convolutions and pooling operations.
* **Attention Mechanisms:** Explored the mathematical foundations of **Transformer attention blocks**, breaking down query, key, value vector transformations and scaled dot-product mechanics.

### 3. Debugging & Evaluation
* **Hyperparameter Tuning:** Systematically analyzed learning rates, initialization schemes, and optimization variants.
* **Architectural Debugging:** Handled gradient verification (gradient checking), vanishing/exploding gradient bottlenecks, and internal shape validation.
* **Model Evaluation:** Built robust evaluation workflows to track loss convergence and metric performance across custom pipelines.

---

## Core Core Mathematical Concepts Covered

* **Linear Algebra:** Matrix decompositions, vector spaces, tensor transformations, and high-dimensional projections.
* **Multivariate Calculus:** Jacobian matrices, chain rule propagation over complex computational graphs, and partial derivatives.
* **Optimization Theory:** Gradient Descent variants, learning rate schedules, and loss surface landscapes.
